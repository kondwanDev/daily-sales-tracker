# Development Roadmap

## Project Goal

Develop a backend application that enables small shop operators to record sales digitally, automate daily sales calculations, and provide reliable sales records.

---

# Milestone 1: Project Setup ✅

## Objectives

* Initialize the project.
* Configure the development environment.
* Prepare project documentation.
* Design the database.

## Tasks

* [x] Create project structure.
* [x] Create virtual environment.
* [x] Install project dependencies.
* [x] Configure FastAPI.
* [x] Initialize Git repository.
* [x] Create project documentation.
* [x] Design the database.
* [x] Implement the PostgreSQL schema.
* [x] Test the database manually.
